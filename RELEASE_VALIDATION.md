# EasyNMT v1.0 Beta — перевірка релізу

Дата перевірки: 2026-07-19.

## Перевірено

- усі Python-модулі компілюються;
- усі JavaScript-файли проходять синтаксичну перевірку Node.js;
- усі 31 Jinja-шаблони завантажуються без помилок;
- усі 6 CSS-файлів проходять парсинг;
- основні сторінки, `/health` і `/ready` не повертають серверних помилок;
- JSON endpoint AI Викладача працює у демо-режимі без ключа;
- SSE endpoint повертає `delta` та завершальний `done` event;
- адаптер нативного Responses API streaming перевірено через контрольований mock;
- справжні PNG приймаються, підроблені файли відхиляються;
- працюють ліміт фото, серверна історія, feedback та видалення приватних вкладень;
- історія поточної розмови береться із серверної бази після першої синхронізації;
- міграція старої AI-таблиці повідомлень зберігає дані та ізолює однакові ID між користувачами;
- OpenAI SDK імпортується лише в `easynmt_ai/service.py`;
- реліз не містить `.env`, локальної бази, завантажених фото, `__pycache__` або `.pyc`.

## Режим без ключа

Сайт запускається без `OPENAI_API_KEY`. AI Викладач переходить у демо-режим, тому підключення ключа можна робити наступним окремим етапом у `EasyNMT v1.0 Beta`.
