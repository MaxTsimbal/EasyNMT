# Easy Chat v3.0 Nebula

Дата: 2026-07-19

## Мета релізу

Завершити великий редизайн чату перед інтеграцією OpenAI. Основний фокус цього релізу — телефон: компактність, відсутність автоматичного зближення Safari, стабільна робота клавіатури та модальні вікна як окремі діалоги.

## Що змінилося

- Мобільний header зменшено до 56 px.
- Великий hero на телефоні замінено компактним привітанням.
- Швидкі дії тепер розташовані у сітці 2×2.
- Composer став однорядковим dock-блоком: вкладення, поле, відправлення.
- Поля вводу на телефоні мають 16 px, тому iOS Safari та Telegram WebView не повинні автоматично масштабувати сторінку.
- `visualViewport` відстежує видиму висоту екрана під час появи клавіатури.
- Rename/Delete відкриваються по центру як окремі вікна без zoom/scale ефекту.
- Збережені всі функції v2.0: локальна історія, пошук, pin, rename, delete, export, streaming, stop, markdown, формули та режими відповіді.
